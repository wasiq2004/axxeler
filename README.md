# Axxeler Static Website

## Structure

- `index.html` - homepage
- `about.html`, `blog.html`, `blog-*.html`, `casestudies.html`, `community.html`, `contact.html` - public pages
- `pages/admin/index.html` - administrative dashboard UI
- `assets/css/` - shared and admin stylesheets
- `assets/js/` - shared and admin scripts
- `assets/images/brand/` - Axxeler brand assets
- `assets/images/clients/` - client logo assets
- `assets/images/awards/` - award imagery
- `assets/images/team/` - team imagery
- `assets/docs/` - downloadable PDFs and documents

## Notes

- The global navigation follows the requested hierarchy: Home, Blog, Community, then other pages.
- The Process navigation item has been removed.
- Homepage FAQ is the final section before the footer, with the primary CTA placed above it.
